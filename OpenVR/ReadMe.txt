ReadMe for ArcGIS Runtime SDK for Android - AR/VR beta 2, rev 1

This package contains: 

ArcGISRuntimeSDKAndroid-ARVR-guide.pdf - Developer's guide for the ArcGIS Runtime SDK for Android beta 2 product.  The guide includes an overview, system requirements, installation details, getting started code examples, and API reference diagrams. 

Libs folder - ArcGIS Runtime SDK for Android beta 2 install libs (aar and pom). See the Guide document for installation details. 

ARVRDemo-Android folder - Android project that showcases simple examples of both AR and VR solutions.
